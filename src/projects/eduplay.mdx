## Overview

    EduPlay is a WordPress development project built to create a fun, informative, and user-friendly website for an educational toy store. The objective was to design an Elementor template kit that combines playful visual appeal with practical eCommerce functionality, making it easy for parents and educators to explore and purchase educational products.
