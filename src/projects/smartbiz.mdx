## Overview

    SmartBiz is a WordPress development project designed to support businesses in the AI and automation industry. The objective was to build an Elementor template kit that communicates innovation, simplifies complex services, and provides a clear path to client engagement. The project blends a tech-forward visual identity with performance-focused design.
