## Overview

    GreenFarms is a WordPress development project built to support organic food retailers with an engaging and easy-to-navigate website. The aim was to create an Elementor template kit that reflects eco-conscious values, promotes clean eating, and enhances the online shopping experience for health-focused consumers.
